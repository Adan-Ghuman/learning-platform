# Learning Platform UI (Frontend)

This is the front-end for the Learning Platform, built with React, TypeScript, and Vite.

## Prerequisites
* Node.js (v18+)
* npm

## Setup Instructions

1. Navigate to the frontend directory:
   cd learning_platform_fe
2. Install dependencies:
   npm install
3. Start the development server:
   npm run dev

The application will be accessible at http://localhost:5173.

## Notes
Ensure the backend API is running concurrently on http://localhost:3000 for full functionality.
